# Sketch Guides
Easy to set exact coodinates "Guides" arround element.

**Sketch 40.^ support**

## Installing Plugins
1. Download and unzip
2. Double click `Sketch Guides.sketchplugin`.
3. Have fun with Sketch Guides.
